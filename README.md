## Search PyPi packages

Open Zazu and type `pypi`

## Installing (MAC)

Add `afaur/zazu-pypi` inside the plugins block of your `~/.zazurc.json` file

## Installing (WIN)

Add `afaur/zazu-pypi` inside the plugins block of your `C:\Users\UserName\.zazurc.json` file.

~~~ javascript
{
  "plugins": [
    'afaur/zazu-pypi'
  ]
}
~~~
